# Engine API Reference

The Backtest360 engine exposes a RESTful JSON API at `https://api.backtest360.com`. All endpoints require an `X-API-Key` header.

The reference below is rendered from the v0.5.3 OpenAPI specification. Admin endpoints (`/admin/*`) are not part of the public surface and do not appear here.
