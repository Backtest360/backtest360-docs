# Engine API Reference

The Backtest360 engine exposes a RESTful JSON API at `https://api.backtest360.com`. All endpoints require an `X-API-Key` header.

The reference below is rendered from the v0.8.13 OpenAPI specification. It covers the strategy, backtest, and analysis endpoints used by the Python client. Internal data-management and administrative endpoints are not part of the public API and do not appear here.
